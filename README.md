# VPN Telegram Bot

## Структура проекта

```
vpn_bot/
├── main.py              # Точка входа
├── config.py            # Загрузка переменных окружения
├── database.py          # SQLite база данных
├── keyboards.py         # Клавиатуры
├── handlers_user.py     # Хэндлеры для пользователей
├── handlers_admin.py    # Хэндлеры для администратора
├── xui_client.py        # Клиент для 3x-ui API
├── requirements.txt     # Зависимости
├── .env.example         # Пример конфига
└── .env                 # Ваш конфиг (создать вручную)
```

## Быстрый запуск

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка .env

Скопируйте `.env.example` в `.env` и заполните:

```bash
cp .env.example .env
nano .env
```

Обязательные поля:

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен от @BotFather |
| `ADMIN_ID` | Ваш Telegram user_id (узнать у @userinfobot) |
| `ADMIN_PASSWORD` | Пароль для входа в админку |
| `SUPPORT_USERNAME` | Username поддержки (без @) |
| `PAYMENT_DETAILS` | Реквизиты для оплаты |
| `XUI_URL` | URL панели 3x-ui |
| `XUI_USERNAME` | Логин от панели |
| `XUI_PASSWORD` | Пароль от панели |
| `XUI_API_TOKEN` | API токен из панели |
| `XUI_HOST` | IP сервера |
| `XUI_PORT` | Порт сервера |

### 3. Настройка inbound в 3x-ui

Откройте файл `xui_client.py` и проверьте `DEFAULT_INBOUND_ID = 1`.

Узнать ID вашего inbound:
1. Зайдите в панель: `https://38.135.55.114:47506/gzlo4U8pXpOQV9287y`
2. Перейдите в раздел Inbounds
3. Скопируйте ID нужного inbound и вставьте в `xui_client.py`

### 4. Запуск

```bash
python main.py
```

## Команды бота

### Пользователь
- `/start` — главное меню

### Администратор
- `/admin` — вход в админку (требует пароль)
- `/orders` — список ожидающих заявок
- `/logout` — выйти из админки

---

## Флоу заказа

```
Пользователь выбирает тариф
        ↓
Бот показывает реквизиты
        ↓
Пользователь оплачивает и отправляет скриншот
        ↓
Админу приходит уведомление с фото и кнопками [Одобрить] [Отклонить]
        ↓
Одобрить → создаётся клиент в 3x-ui → ссылка отправляется пользователю
Отклонить → админ пишет причину → причина отправляется пользователю
```

## Цены
